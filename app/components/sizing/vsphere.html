Minimum Resources for vSphere Cluster

Your vSphere cluster must have the following minimum resources before you install Pivotal Cloud FoundryÂ®

  • Disk space: 1TB
  • Memory: 120GB
  • Two public IP addresses: One for Elastic Runtime and one for Ops Manager
  • vCPU cores: 80
  • Overall CPU: 28 GHz
  • 1 TB of storage
  • Ops Manager Director must have HTTPS access to vCenter and ESX hosts on TCP ports 443, 902, and 903.
