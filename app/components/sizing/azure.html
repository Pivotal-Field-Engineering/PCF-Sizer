Minimum Resources on Azure

Your Azure account must have the following resources before you install Pivotal Cloud Foundry®

  • Pay-as-you-go subscription on your Azure account.
  • 1 Storage Account
  • Sufficiently high quota limits (Recommended: 100 cores) on your Azure account.
  • Installing Pivotal Cloud Foundry® requires more than the default 20 cores
  • 3 Public IPs
  • 1 “jumpbox” VM that runs the deployment using BOSH
  • The deployment uses a total of 53 VMs, of which 38 are up and running at the end of the deployment. The other 15 VMs are created to run jobs and errands and are subsequently destroyed
